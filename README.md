# Hazometric

**The little brother of Haze** – Hazometric is a lightweight game launcher that allows you to organize and run your games easily.

---

## Features

* Add games to your library with custom titles and descriptions
* Launch games directly from the app
* Remove games from the library
* Organize games in tabs for easy navigation
* Scrollable library view for multiple games
* Supports saving and loading your library between sessions

---

## Usage

### Running the App

The main window will open with:

* **Library Scroll Frame** on the left
* **Tab View Frame** on the right
* **Utility buttons** at the bottom

---

### Adding a Game

1. Click **“Add to library”**
2. Enter the following details:

   * **Path**: Path to the `.exe` or other executable
   * **Title**: Name of the game
   * **Description**: Short description of the game
3. Click **Confirm**
4. Your game will appear as a card in the scrollable library
5. Click the game card to open its **game tab**, with options:

   * **Play** – Launch the game
   * **Close this tab** – Close the game tab
   * **Remove from library** – Remove the game

---

### Managing Games

* To remove a game, click **Remove from library** inside its game tab.
* To close a game tab without removing it, click **Close this tab**.
* You can switch between games using the **tabs** on the right.

---

### File Saving

* Game library is stored in `save/userData.json`
* Hazometric automatically loads this data at startup
* No manual editing is needed
